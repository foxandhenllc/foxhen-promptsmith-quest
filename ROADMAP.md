# Promptsmith Quest Roadmap

This roadmap keeps Promptsmith Quest focused on practical open-source use: A playable prompt-building puzzle for assembling constraints, quality gates, scoring, and exportable prompt handoffs.

## Near-Term

- Add tutorial polish, keyboard coverage, and a clearer exported run summary.
- Add smoke coverage for the primary workflow and exported output.
- Improve accessibility coverage for labels, focus states, keyboard flow, and mobile layouts.
- Add more screenshots or example exports that reviewers can inspect without running the app.
- Tighten fork/setup docs so a new contributor can make one useful change in under an hour.

## Mid-Term

- Add reusable scenario packs, templates, or fixtures that can be copied into a fork.
- Publish a small example gallery showing realistic public-safe adaptations.
- Add stricter content validation for docs, links, and sample data.
- Improve release notes and versioned handoff examples.

## Non-Goals

- No private customer data in the repository.
- No account system or hosted private workspace.
- No tracking or analytics requirement.
- No opaque external enrichment service.
- No production deployment assumptions beyond a static web build unless explicitly documented.
